## ⚙️ **Premissas do Projeto**
As análises realizadas consideraram as seguintes premissas:
- **Confiabilidade dos Dados**: Acreditamos que os dados fornecidos são precisos e representativos.
- **Tendências Sazonais**: As variações de vendas refletem sazonalidades reais.
- **Preferências por Região**: Os comportamentos de compra variam entre os países analisados.
- **Capacidade Logística Atual**: A capacidade de transporte e estoque está alinhada aos dados fornecidos.

<br>

## 🏗️ **Estrutura do Projeto**
O projeto foi desenvolvido em três fases principais:

### 🚀 **Fase 1: Coleta e Limpeza de Dados**
- Os dados foram extraídos de arquivos como `Meganium_Sales_Data_-_Consolidado.xlsx`.
- Valores nulos foram removidos, e colunas com formatos inadequados foram ajustadas.

### 📊 **Fase 2: Análise Exploratória e Visualização**
- Identificação de produtos mais populares.
- Criação de planilhas que mostram sazonalidades e insights regionais.

### 💡 **Fase 3: Extração de Insights**
- Geração de recomendações estratégicas com base em dados visuais e métricas.

<br>

## 🔬 **Metodologia**
O projeto utilizou a seguinte abordagem:

1. **Extração e Carregamento dos Dados**
   - Uso de Python com bibliotecas como `pandas` e `openpyxl`.

2. **Limpeza dos Dados**
   - Padronização de formatos e exclusão de valores inconsistentes.

3. **Visualizações e Gráficos**
   - Criação de gráficos usando `matplotlib` e `seaborn`.

4. **Geração de Insights**
   - Análise cruzada dos dados para extrair padrões e recomendações.

<br>

## 📈 **Resultados Finais**
### 🌟 **Principais Descobertas**
- **Produto mais popular globalmente**: NEW MEGANIUM RG 40XXV.
- **Pico de vendas**: Agosto, associado a sazonalidade e campanhas.
- **Mercados estratégicos**: Canadá, França e Japão apresentaram maior receita.

### 📈 **Recomendações**
1. **Campanhas Regionais**: Expandir promoções específicas para mercados prioritários.
2. **Ajustes Logísticos**: Reduzir tempos de entrega nos mercados com maior potencial.
3. **Otimização de Estoques**: Garantir alta disponibilidade dos produtos mais vendidos antes do pico de agosto.
